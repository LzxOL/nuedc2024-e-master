# nuedc2024-e

#### 介绍
湖北省电赛2024年E题三子棋代码，在六道题中做出1、2、4、5问，4/6获得湖北省2024年 **省级二等奖**，学校唯二 。
第六题由于时间不够充分，在上位机的代码已经写好的情况下，没有能够调试出来，略微遗憾。
